# Configuration Example Files

Use one file at a time as the basis for your repository config.

## How to Use

1. Copy one example.
2. Rename it to release-workflow.yml.
3. Place it in the repository config path expected by Probot.
4. Commit and test with milestone.created events.

## Available Variations

- release-workflow.basic-init.yml
- release-workflow.init-plus-project-item.yml
- release-workflow.conditional-release-title.yml
- release-workflow.staged-rollout-disabled-task.yml
- release-workflow.multi-automation-first-enabled.yml
- release-workflow.invalid-unsupported-action.yml
